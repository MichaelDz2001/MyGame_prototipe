# Overview
A Python prototype game made with Pygame. The current version includes only a menu (no gameplay yet).

# Requirements
 - Python 3
 - pygame
Install:
pip install pygame

# How to Run
From the repository root:
python game.py

# Project Files
 - game.py — entry point / menu
